### App Hotel feito em .NET Maui

#### App criado nas aulas da Fatec Jahu, neste projeto, entendemos conceitos novos dentro das contentPage, aprendemos o início de bindingContext, e novas sintaxe de marcação para estilizar melhor nossas telas.

#### FEATURES DO PROJETO

- BindingContext present para repassar informações de uma View a outra
- Security storage para salvar valores até o final do ciclo de vida da aplicação
- Autenticação*

*validação sem persistencia de dados, apenas ambiente sandbox.
